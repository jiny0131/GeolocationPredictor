This README describes the various files submitted for 2021S1 COMP90049 Project 3.

There are two files about code, the build_network and a3 file. And one file about report, the IML report file.

FOR THE CODE PART

The build_network file is used to create node2vec feature by the raw data file provided.
- Input: full Tweets files (train_full, dev_full, test_full)
- Output: Node2vec feature files with name (train_node2vec300, dev_node2vec300, test_node2vec300)

The a3 file is mainly about the architecture of three different models and one 0-R baseline.
